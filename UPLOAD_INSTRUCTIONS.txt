## Important: Update the Documentation Link

In `index.html`, find this line (near the end of the file):

```javascript
const repoUrl = 'https://github.com/YOUR-USERNAME/vehicle-log/blob/main/SETUP_GUIDE.md';
```

**Replace `YOUR-USERNAME` with your actual GitHub username.**

For example, if your GitHub username is `pauljones`, change it to:
```javascript
const repoUrl = 'https://github.com/pauljones/vehicle-log/blob/main/SETUP_GUIDE.md';
```

This makes the Help button link to your actual documentation!
